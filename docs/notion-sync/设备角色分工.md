## 关键注意事项
1. Mac 装 rosbags / lerobot 偶尔卡 native 编译 → 立刻切无影 Ubuntu，别死磕
1. ECS 默认 40GB 系统盘易被数据集打爆 → 下载前 df -h 检查，不够就挂 100GB 数据盘（按量几块钱/月）
1. 录屏只用 Mac（Kap/QuickTime）→ iPhone 录屏分辨率/字号都不适合做技术 demo
1. H50 前必做 GPU 实例开机 → 拉好 Pi05 VLA 权重 + LIBERO 镜像，否则 H54 才开始会再吞 2-3h

## 数据下载推荐路径
```bash
# ECS 下载（速度比 Mac 直连快 3-5×）
ssh ecs-host
export HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download --repo-type dataset lerobot/svla_so100_pickplace \
    --local-dir ~/datasets/so100_pickplace

# Mac 拉回
rsync -avz --progress ecs-host:~/datasets/so100_pickplace ./datasets/
```
